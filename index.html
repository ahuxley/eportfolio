Featured Project

MicroFighter

MicroFighter is a 2D two-player fighting game that I built in Godot. What made the project especially interesting was that we did not stop at the game itself. We also built a custom Arduino Mega R3 controller using push buttons and an analog stick.

To make the hardware work with the game, I wrote a Python bridge that reads Arduino serial input, interprets button masks and analog stick values, and converts them into player controls that the game can use. The script maps inputs for two players, handles analog deadzones, and can send both Windows keyboard events and UDP messages for integration.

  • Built gameplay in Godot for a local two-player fighting experience
  • Helped create a custom Arduino Mega R3 controller with push buttons and an analog stick
  • Wrote a Python bridge that parses serial payloads, maps player actions, and translates hardware input into usable game controls

The screenshots on the left show the room-pricing menu and the reservation flow using the real program output from this project.
