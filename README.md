# Andrew Huxley Portfolio

This is my personal portfolio website.

Live site: `https://ahuxley.github.io/eportfolio/`

## Built with

- HTML
- CSS
- JavaScript
- GitHub Pages

## Run locally

```bash
python -m http.server 4173
```

Then open `http://localhost:4173`.

## Contact form

The contact form can use Formspree.  
If no Formspree endpoint is configured, it falls back to opening a prefilled email draft.

Set endpoint in `index.html`:

```html
<meta name="formspree-endpoint" content="https://formspree.io/f/your-form-id">
```
